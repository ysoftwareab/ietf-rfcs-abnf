1. Introduction

    To implement an html syntax in index.html or equiv that can power on or
    off website

2. Background
     
    You are publisher of a website. You should be able to power on or off the 
    website to everyone or selective regions using index.html or equivalent
3. General Information of the service
	
    If i can say that only those in bangkok and amsterdam and sanfrancisco
    should be able to view my website through an html or if i cn say its powered
    on or off for everyone using an html syntax
