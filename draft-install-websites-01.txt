                 USer Invokable Actions on Desktops to install/copy Websites
                         draft-install-websites-01

Abstract

    The document describes the need for a User Invokabke Action in Desktops
    of Windows , Android and many other Operating systems bundled on mobile
    computing devices to install or copy a website for offline reading from
    a filesystem.
